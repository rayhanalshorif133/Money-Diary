# 💰 Money-Diary

**Money-Diary** is a simple yet powerful personal finance tracker that helps you record and manage your **daily expenses, income, and financial activities**.  
The goal of this project is to provide a lightweight, browser-based solution with **Supabase as the backend** for secure data storage.

---

## 🚀 Features

- ✍️ **Daily Expense Logging** – Record your day-to-day transactions easily  
- 💵 **Income Tracking** – Keep track of all your revenue sources  
- 📊 **Share/Stock Calculation** – Calculate profit/loss from trades or investments  
- 📈 **Visual Insights** – Get summary reports for better financial decisions  
- ☁️ **Cloud Storage** – Securely store and sync data with **Supabase**  
- 📱 **Responsive Design** – Works across devices (mobile & desktop)  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, jQuery  
- **Backend / Database:** Supabase (PostgreSQL, Auth, API)  
- **Other Tools:** Supabase JS SDK
